# SHMRY

Infrastructure telemetry, governance, and automation — unified.

SHMRY is a lightweight infrastructure intelligence platform for companies that need operational visibility, auditability, and policy-governed automation.

## Core Features

- Infrastructure health monitoring
- Telemetry visibility
- Billing analytics
- Audit history
- Regional topology mapping
- Policy-governed automation
- Git-backed governance

## Editions

### Community Edition
- Healthchecks
- Telemetry
- Logs
- SQLite runtime state

### Business Edition
- Automation
- Billing analytics
- Governance controls
- Operational dashboards

### Sovereign Edition
- Private deployment
- Regional governance
- Compliance workflows
- Infrastructure digital twin
